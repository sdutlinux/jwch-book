母盘制作过程
---------------------

#. 安装win xp 系统
#. 安装补丁包
#. 安装驱动
#. 安装软件
#. 关闭防火墙
#. 关闭自动更新
#. 管理员用户名为 admin
#. 开启远程桌面
#. 设置远程桌面无密码登陆,在组策略里设置,
#. 设置 web 桌面,
#. 设置网络硬盘,
#. 用360忧化,关闭不需要的启动项,服务
#. 清理垃圾,缓存(360可以)
#. 安装小哨兵网络版驱动。
#. 查年小哨兵是否能连接上服务器(360设置可能有关)
#. 关闭360弹窗
#. ghost 备份
