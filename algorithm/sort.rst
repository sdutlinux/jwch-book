堆排序 Heap Sort
-----------------


最大堆：所有节点的子节点比其自身小的堆。
最小堆：所有节点的子节点比其自身大的堆。

用数组模拟二叉树（当然也包括堆）的话，如果根节点的下标为0的话，则对于每个结点i，其左孩子下标为2*i+1；其右孩子下标为2*i+2。

堆有两个基本操作：插入元素和删除元素。

堆排序在线演示 : http://www.cs.usfca.edu/~galles/visualization/HeapSort.html
